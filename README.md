# ClassifiedsApp

Classifieds is a market place for trading of used goods within the college community. Users can buy or sell used goods. The application is made for Android OS using android SDK And firebase. 

## Functionalities:
### Authentication: Users can create an account and login into it.
![signup](https://user-images.githubusercontent.com/72807512/142972227-25c75591-cbdc-4ebf-8b6d-d9e4bfe762bc.jpeg)

----


### Feed: Users can browse products and click on them to open a product page and view the details.
![feedpage](https://user-images.githubusercontent.com/72807512/142971004-59147ee2-0080-46be-83ef-b806a9fffb71.jpeg)

----


### Post Product: Users can post a product they want to sell by putting up an image and details of the product. 
![post page](https://user-images.githubusercontent.com/72807512/142971467-4cae9a1c-eeb6-4f94-b8df-f26b02aea159.jpeg)

----

### Chat/Call: The user can get in touch with the seller by either using in-app chat services or by calling them.  
![productPage](https://user-images.githubusercontent.com/72807512/142971622-55a7a10b-ad7c-461c-90f7-46d3d6a6bba1.jpeg)
![chat Interface](https://user-images.githubusercontent.com/72807512/142971783-5b461d05-9661-4530-b878-4b1802316aaf.jpeg)

----

### Account Section: Users can edit their account details.
